# 🌱 achitree

## 🎮 Controls

### 🖱️ Mouse
- **Left Click** → Build new root/branch node  
- **Left Click + T** → Build new trunk node  
- **Right Click** → Move camera  
- **Scroll Wheel** → Zoom camera  

### 🎯 Focus Shortcuts
- **D** → Focus to default  
- **L** → Focus to light  
- **W** → Focus to water  
- **M** → Focus to mineral  

### 🍬 Resources & Actions
- **S** → Add 1 sugar order  
- **Shift + S** → Remove 1 sugar order  

### 🛠️ Crafting
- **U** → Craft new unit *(costs 1 sugar)*  
- **B** → Craft bud *(costs 1 sugar + 20 water + 20 mineral)*  

### 🔄 System
- **R** → Restart game  
- **Esc / Alt + F4** → Close game
